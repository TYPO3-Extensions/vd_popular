List of markers

  • LIST : Begin & end of the page list
    • ITEM : Begin & end of the items
      • PAGE_TITLE : Link to the page. The text is the name of the page
      • PAGE_NAVTITLE : Link to the page. The text is the navigation name of the page
      • PAGE_DESCRIPTION : Page description
      • LINK : Link wrap
      • COUNTER : Number of visits for the recording perdiod define in the extensions settings
      • TEXT1 : Some text (-> pi2/locallang.xml)
      • TEXT2 : Some text (-> pi2/locallang.xml)
  • NO_RECORDS : Begin & end of the message if there is no records
    • NO_RECORDS_TEXT : Text message if no records (-> pi2/locallang.xml)
